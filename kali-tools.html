الرئيسية أدوات Kali Linux المصادر

أدوات Kali Linux

دليل تفصيلي لجميع أدوات Kali Linux المصنفة حسب الاستخدام، مع شرح مفصل لكل أداة وكيفية استخدامها

أدوات جمع المعلومات

Nmap

Network Mapper

أداة قوية لاكتشاف الشبكات وفحص المنافذ، تستخدم لاكتشاف الأجهزة المتصلة بالشبكة والخدمات المفتوحة وتحديد أنظمة التشغيل.

الاستخدامات:

  • • اكتشاف الأجهزة على الشبكة
  • • فحص المنافذ المفتوحة
  • • تحديد أنظمة التشغيل
  • • اكتشاف الخدمات والإصدارات

أمثلة الأوامر:

# فحص أساسي
nmap 192.168.1.1

# فحص شامل مع تحديد الخدمات
nmap -sV -sC 192.168.1.1

# فحص جميع المنافذ
nmap -p- 192.168.1.1
الصعوبة: متوسطة مجاني ومفتوح المصدر

theHarvester

OSINT Tool

أداة لجمع المعلومات من المصادر المفتوحة (OSINT) لجمع البيانات عن النطاقات وعناوين البريد الإلكتروني وغيرها.

الاستخدامات:

  • • جمع عناوين البريد الإلكتروني
  • • اكتشاف النطاقات الفرعية
  • • جمع عناوين IP
  • • البحث عن أسماء الموظفين

أمثلة الأوامر:

# البحث عن نطاق معين
theharvester -d example.com -b google

# البحث باستخدام جميع المصادر
theharvester -d example.com -b all
الصعوبة: سهلة مجاني ومفتوح المصدر

Recon-ng

Web Reconnaissance Framework

إطار عمل قوي لجمع المعلومات من الويب يوفر واجهة سطر الأوامر وقاعدة بيانات لإدارة المعلومات المجمعة.

الاستخدامات:

  • • إنشاء قواعد بيانات للهدف
  • • استخدام وحدات متعددة للبحث
  • • تخزين النتائج بشكل منظم
  • • دعم مصادر متعددة

أمثلة الأوامر:

# بدء Recon-ng
recon-ng

# إنشاء مساحة عمل
workspaces create target_domain

# تحميل وحدة بحث
modules load recon/domains-hosts/google_site_web
الصعوبة: متوسطة مجاني ومفتوح المصدر

Maltego

Data Mining Tool

أداة تعدين بيانات مرئية لتحليل العلاقات بين الكيانات المختلفة وعرضها في رسوم بيانية تفاعلية.

الاستخدامات:

  • • رسم خرائط العلاقات
  • • تحليل البنية التحتية
  • • اكتشاف الروابط بين الكيانات
  • • تحليل الشبكات الاجتماعية

المميزات:

  • • واجهة رسومية سهلة الاستخدام
  • • دعم مصادر متعددة
  • • عرض تفاعلي للبيانات
  • • تصدير النتائج بصيغ متعددة
الصعوبة: سهلة نسخة مجانية ومدفوعة

أدوات تحليل الثغرات

Nikto

Web Vulnerability Scanner

محلل ثغرات ويب شامل يقوم بفحص خوادم الويب للكشف عن الملفات الخطرة والبرامج القديمة وغيرها من المشكلات.

الاستخدامات:

  • • فحص خوادم الويب
  • • اكتشاف الملفات الخطرة
  • • التحقق من الإصدارات القديمة
  • • فحص إعدادات الخادم

أمثلة الأوامر:

# فحص موقع ويب
nikto -h http://example.com

# فحص مع تجاوز IDS
nikto -h example.com -evasion 1234
الصعوبة: سهلة مجاني ومفتوح المصدر

OpenVAS

Vulnerability Assessment System

نظام متكامل لتحليل الثغرات يوفر فحصاً شاملاً للشبكات والأنظمة مع تقارير مفصلة عن الثغرات المكتشفة.

الاستخدامات:

  • • فحص الشبكات الشامل
  • • تحليل الثغرات الأمنية
  • • إنشاء تقارير مفصلة
  • • إدارة الثغرات

المميزات:

  • • قاعدة بيانات ضخمة للثغرات
  • • واجهة ويب سهلة الاستخدام
  • • تقارير قابلة للتخصيص
  • • تحديثات منتظمة
الصعوبة: متوسطة مجاني ومفتوح المصدر

Lynis

Security Auditing Tool

أداة تدقيق أمني لأنظمة Unix/Linux تقوم بفحص النظام وإعطاء توصيات لتحسين الأمان.

الاستخدامات:

  • • تدقيق أمني للنظام
  • • فحص إعدادات الأمان
  • • اختبار الامتثال
  • • تحسين الأمان

أمثلة الأوامر:

# فحص النظام
lynis audit system

# فحص سريع
lynis --quick
الصعوبة: سهلة مجاني ومفتوح المصدر

Wapiti

Web Application Vulnerability Scanner

محلل ثغرات تطبيقات الويب يكتشف الثغرات الشائعة مثل XSS وSQL Injection وغيرها.

الاستخدامات:

  • • فحص تطبيقات الويب
  • • اكتشاف XSS وSQL Injection
  • • فحص أمن النماذج
  • • اختبار الاختراق السوداء

أمثلة الأوامر:

# فحص موقع ويب
wapiti -u http://example.com

# فحص مع تقرير HTML
wapiti -u http://example.com -f html -o report.html

أدوات الاستغلال

Metasploit Framework

Penetration Testing Framework

إطار عمل شامل لاختبار الاختراق يحتوي على آلاف الثغرات والحمولات المختلفة لاختبار أنظمة متعددة.

الاستخدامات:

  • • استغلال الثغرات الأمنية
  • • إنشاء حمولات خبيثة
  • • اختبار ما بعد الاختراق
  • • إدارة جلسات الاختراق

أمثلة الأوامر:

# بدء Metasploit
msfconsole

# البحث عن ثغرة
search ms17-010

# استخدام ثغرة
use exploit/windows/smb/ms17_010_eternalblue

Burp Suite

Web Application Security Testing

منصة متكاملة لأمن تطبيقات الويب توفر وكيل ويب، محلل ثغرات، وأدوات يدوية لاختبار الأمان.

الاستخدامات:

  • • اعتراض طلبات HTTP/HTTPS
  • • فحص تطبيقات الويب
  • • اختبار SQL Injection وXSS
  • • تلاعب بالطلبات

المميزات:

  • • واجهة رسومية سهلة الاستخدام
  • • دعم واسع لأنواع الثغرات
  • • تقارير مفصلة
  • • إضافات متعددة
الصعوبة: متوسطة نسخة مجانية ومدفوعة

أدوات الشبكات اللاسلكية

Aircrack-ng

Wireless Security Assessment

مجموعة أدوات متكاملة لاختبار أمن الشبكات اللاسلكية وفك تشفير كلمات مرور WEP/WPA/WPA2.

الاستخدامات:

  • • فك تشفير WEP/WPA/WPA2
  • • مراقبة حركة المرور اللاسلكية
  • • هجمات إلغاء المصادقة
  • • تحليل حزم البيانات

أمثلة الأوامر:

# تفعيل وضع المراقبة
airmon-ng start wlan0

# التقاط الحزم
airodump-ng wlan0mon

# فك التشفير
aircrack-ng capture.cap -w wordlist.txt
الصعوبة: متوسطة مجاني ومفتوح المصدر

Reaver

WPS Attack Tool

أداة متخصصة في هجمات WPS (Wi-Fi Protected Setup) لفك تشفير كلمات مرور الشبكات اللاسلكية.

الاستخدامات:

  • • هجمات WPS PIN
  • • فك تشفير WPA/WPA2
  • • اختبار أمان WPS
  • • هجمات التخمين

أمثلة الأوامر:

# هجوم WPS
reaver -i wlan0mon -b XX:XX:XX:XX:XX:XX -vv

# هجوم مع تجاهل الإخفاقات
reaver -i wlan0mon -b MAC -N -S -vv
الصعوبة: متوسطة مجاني ومفتوح المصدر

أدوات كلمات المرور

John the Ripper

Password Cracker

أداة قوية لفك تشفير كلمات المرور تدعم أنواعاً متعددة من التجزئة والتشفير.

الاستخدامات:

  • • فك تشفير كلمات المرور
  • • اختبار قوة كلمات المرور
  • • هجمات القاموس والقوة الغاشمة
  • • دعم أنواع متعددة من التجزئة

أمثلة الأوامر:

# فك تشفير ملف
john hash.txt

# استخدام قاموس محدد
john --wordlist=passwords.txt hash.txt
الصعوبة: متوسطة مجاني ومفتوح المصدر

Hashcat

Advanced Password Recovery

أداة متقدمة لفك تشفير كلمات المرور تستخدم GPU لتسريع العملية، وتدعم أنواعاً كثيرة من التجزئة.

الاستخدامات:

  • • فك تشفير باستخدام GPU
  • • هجمات متعددة الأنواع
  • • دعم 300+ نوع تجزئة
  • • أداء عالي جداً

أمثلة الأوامر:

# فك تشفير MD5
hashcat -m 0 hash.txt passwords.txt

# استخدام GPU
hashcat -m 0 hash.txt passwords.txt -O
